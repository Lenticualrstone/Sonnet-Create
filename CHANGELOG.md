# Changelog

## v1.3.0 (개발 중)

AI 에이전트를 진짜 창작 파트너로 끌어올리고, 인터페이스를 백색·딥네이비 단일 테마로 재정의한 업데이트.

### AI 에이전트 🤖
- **4개 AI 프로바이더 지원** — Claude / ChatGPT / Gemini / Grok API 연결. 설정 > Sonnet AI에서 프로바이더 선택, 프로바이더별 API 키(Keychain 보관)와 모델 선택(기본 모델 또는 추천 목록).
- **실시간 스트리밍 응답** — 채팅 응답이 (연출이 아닌) 실제 API 스트리밍으로 도착 즉시 표시된다. 모든 클라우드 프로바이더 공통.
- **에이전트 페르소나** — 에이전트 이름과 행동지침 페이지(마크다운)를 설정하면 모든 대화·문서 생성의 시스템 프롬프트에 합성된다.
- **Tool Use — 에이전트가 앱을 직접 조작** 🛠️ — 이제 대화만으로 앱 기능을 시킬 수 있다. "내 프로젝트 뒤져서 주인공 설정 참고해 새 장면 써줘" 같은 요청에 에이전트가 스스로 검색 → 읽기 → 생성을 이어서 수행한다. 실행 중인 기능은 채팅에 칩으로 표시돼 무엇을 하는지 감춰지지 않는다.
  - **탐색·읽기**: 프로젝트 목록, 문서 목록(종류/프로젝트 필터), 본문 검색, 문서 읽기, 지금 열린 문서 확인
  - **생성**: 프로젝트, 일반 문서, 캐릭터 문서, 마인드맵(방사형 자동 배치), 시나리오(캐스트 자동 구성)
  - **부분 편집**: 시나리오 블록 추가/수정(원하는 위치에 삽입), 마인드맵 노드·연결 추가/수정, 캐릭터 프로필·보이스 카드 수정(지정한 항목만), 페이지 이어 쓰기, 이름 변경. 문서를 다시 만들지 않고 "이 대사만 고쳐줘"가 된다.
  - **파괴적 작업은 실행 전 확인** ⚠️ — 휴지통 이동, 시나리오 블록 삭제, 마인드맵 노드 삭제, 본문 전체 교체는 **무엇이 사라지는지 먼저 보여주고** 사용자가 승인해야 실행된다. 거부하면 에이전트가 재시도하지 않고 방향을 바꾼다. 확인 절차가 없으면 아예 실행되지 않는 안전 기본값.
  - 편집 대상은 문서 읽기가 알려주는 짧은 핸들(`[a1b2c3d4]`)로 지목한다 — 블록이 추가·삭제돼도 흔들리지 않는다.
  - 도구 정의는 제공자 중립이고 어댑터가 각 사 포맷(Anthropic `input_schema` / OpenAI `function` / Gemini `function_declarations`)으로 번역한다 — **새 기능을 붙일 때 도구 핸들러 하나만 등록하면 4개 프로바이더 전부에 자동 노출된다.**
  - 도구를 지원하지 않는 오프라인/온디바이스 제공자에서는 도구 없이 텍스트 대화로 자연히 떨어진다.
- **에이전트 문서 생성** — 채팅 입력창의 + 메뉴로 브리프를 주면 문서를 통째로 생성해 워크스페이스에 저장하고 연다:
  - **문서 자동 작성** — 마크다운 → 블록 페이지(.scpa)
  - **캐릭터 문서 생성** — 프로필(역할/요약/필드/보이스 카드) + 본문(.scpa character)
  - **마인드맵 생성** — 노드/엣지 설계 + 방사형 자동 배치(.scno)
  - **시나리오 작성** — 캐스트 자동 구성 + 대사/지침 블록(.scen)
  - 어느 프로바이더로도 동작하도록 출력 형식(마크다운/JSON/시나리오 포맷)을 강제하는 파서 내장.
- 시나리오 이어쓰기(자동작성)도 새 프로바이더 전부에서 동작.

### 인터페이스 — 테마 재정의 🎨

## v1.2 "Safety Net" — 2026-07-13

원고를 잃는 모든 조용한 경로를 막고, 대규모 프로젝트 탐색을 다시 설계한 업데이트. 저장은 실패해도 소리 내어 알리고, 손상된 문서는 스냅샷으로 되살아난다.

### 안정성 — "믿고 맡기는 저장"
- **손상 문서 데이터 손실 차단** — content.json이 깨진 문서를 빈 문서로 대체해 열던 동작 제거. 이제 열기를 중단하고 알린 뒤, 번들에 스냅샷이 있으면 **원클릭 복구**(최신 스냅샷으로 세션을 열고 즉시 재저장해 번들 치유)를 제안한다.
- **저장 실패 표면화** — 실패 사유를 상태 배지 툴팁에 표시(클릭=재시도). 저장 실패 상태로 탭을 닫으면 확인 다이얼로그(다시 시도/저장 없이 닫기/취소), 종료 시에는 applicationShouldTerminate 가드로 경고 후 선택.
- **백업 복원 안전화** — 복원 전에 열린 문서 탭을 전부 닫는다 (열린 세션의 자동저장이 복원 전 내용을 도로 덮어쓰던 사고 차단). 백업/복원/scproj 압축·해제를 전부 백그라운드로 이동, 진행 스피너 표시.
- **최근 문서 영속 수정** — recents 저장 키가 실행마다 바뀌는 해시를 쓰고 있어 재시작하면 목록이 사라졌다. 안정 키로 교체.
- **통계 정합성** — undo/redo/스냅샷 복원이 일일 집필량·스트릭에 집계되던 문제 수정. 마인드맵 팬/줌이 문서를 '미저장'으로 만들던 문제 수정.
- **성능** — 저장할 때마다 전 문서 본문을 메인 스레드에서 다시 읽던 색인 갱신을 백그라운드 단일 트랜잭션으로 재설계 (세대 가드로 경합 제거), watcher/자동저장 스캔 디바운스. 문서가 수백 개여도 타이핑이 끊기지 않는다.
- 가리기/휴지통/이름변경이 metadata.json만 갱신 (손상 문서에도 동작), 읽기 전용 탭의 유령 이름변경 차단, 캐스트 연결 캐릭터 페이지 즉시 영속화(끊긴 참조 방지), 검색 LIKE 와일드카드 이스케이프, 열기 실패 시 경고+수신함 이벤트.

### 인터페이스 — 프로젝트 네비게이터 재설계
- **사이드바 평탄화** — 프로젝트 행이 하위 문서를 펼치지 않는다 (대규모 프로젝트에서 목록이 한없이 길어지는 문제 해소). 문서 수 배지 표시, 클릭하면 해당 프로젝트로 필터된 파일 아카이브가 열린다.
- **프로젝트 파일 인스펙터** — 프로젝트 소속 문서를 열면 편집기 우측에 간이 Finder: 종류별(시나리오/마인드맵/페이지/캐릭터/기타 첨부) 목록, 현재 문서 강조, + 메뉴로 프로젝트 안에 새 문서 생성, 하단 '프로젝트 아카이브 열기'. 탭바 폴더 버튼으로 토글.
- **캐릭터 인스펙터 좌측 고정** — 우측은 파일 인스펙터 자리. 위치 설정 옵션 제거.
- **탭 드래그 재정렬** — 탭 칩을 끌어 순서 변경 (라이브 리오더), ⌘1~9도 새 순서를 따른다.
- **백업 타임라인** — 백업별 문서 수·용량 표시 (백그라운드 계산).

### 새 기능
- **대본 내보내기** 🎭 — 시나리오를 무대 대본 형식 텍스트(.txt)와 A4 다중 페이지 PDF로. 등장인물 헤더, 화자·들여쓴 대사, 괄호 지침, 장면 전환 표시. 분기 모드에서는 분기 흐름 기준.
- **씬 목차** — 구분선 블록을 장면 경계로 삼아 툴바 메뉴에 목차(첫 대사 미리보기 포함) 표시, 선택하면 해당 장면으로 점프. 장면 2개 이상일 때 등장.
- **주간 집필 리포트** — 프로필에 최근 7일 미니 막대(목표 달성일은 진한 액센트), 7일 합계, 최고 기록, 지난주 대비 증감.
- **GitHub 릴리스 자동 업데이트** ⬇️ — 실행 시(끄기 가능)·수동으로 최신 릴리스를 확인. 새 버전이 있으면 탭바에 강조색 배지 인디케이터가 뜨고, 퀵메뉴에서 릴리스 노트 확인·다운로드 및 열기(.dmg 자산)·이 버전 건너뛰기. 설정 > 기본에 자동 확인 토글 + 지금 확인.

### 인터페이스 — 심미 · 인터랙션
- **AI 스피어** 🔮 — Sonnet AI의 아이덴티티 오브. 5종 스타일(파티클/글래스/홀로그램/잉크/플라즈마)을 설정에서 라이브 프리뷰로 선택, 커서를 스치면 입자가 밀려나 흩어지고 채팅 입력·생성 중엔 요동친다. 파티클 밀도(성기게/보통/촘촘)도 선택. 채팅 히어로·헤더·시나리오 이어쓰기 인디케이터·사이드바 탭 아이콘에서 공유. (Metal 셰이더 혼용)
- **ASCII 웨이브 히어로** 〜 — 홈 인사말 위 문자 물결. 커서가 지나가면 글자가 밀려나 흩어졌다 돌아오고, 시간대(아침/낮/저녁/심야)에 따라 흐름 속도와 인사말 문구가 바뀐다. 히어로 클릭 = Sonnet AI 열기.
- **강조색 일관성** — 세그먼트 픽커·설정 사이드바·아카이브 카테고리 칩·프로젝트 필터를 앱 강조색을 따르는 커스텀 컨트롤로 교체(시스템 컨트롤이 OS 색을 강제하던 문제). 다크 모드에서 어두운 커스텀 강조색 자동 보정. 활성 탭 언더라인이 탭 사이를 미끄러진다.
- **채팅 응답 타이핑 연출** — 응답이 글자 단위로 점진 노출된다.
- **컨텍스트 인식 새 문서** — 헤더 +·메뉴·⌘K 팔레트·⌘N/⇧⌘N이 현재 보고 있는 프로젝트 안에 문서를 만든다(무소속으로 떨어지던 문제 수정). 대상 프로젝트를 메뉴·툴팁에 표시. 빈 아카이브 카테고리에 원클릭 생성.
- **아카이브 키보드 탐색** — ↑↓로 포커스 이동(강조색 링), ⏎ 열기, ⎋ 해제.
- **탭 드래그 vs 창 이동 분리** — 탭 순서 변경 드래그가 창을 끌던 문제 수정. 프로젝트 파일 인스펙터 폭 드래그 조절·표시 상태 영속.

### 개발 체계
- 손상/부분 손실 회귀 테스트(DocumentKit 3종), 대본 내보내기 테스트(텍스트 형식·PDF 유효성), SwiftPM .build 산출물 5천여 개 추적 해제. 스트리밍 재전송 경합·ASCII 필드 0-나눗셈 등 신규 인터랙션 엣지케이스 하드닝.

## v1.1 "Take Two" — 2026-07-11

되돌릴 수 있는 안전망(스냅샷·복원), 대본이 소리 내어 읽히는 리허설, 어디서든 문서로 점프하는 팔레트 — 창작을 "다시 찍을 수 있게" 만든 업데이트.

### 새 기능
- **⌘K 커맨드 팔레트** — 최근 문서 + 딥서치(제목·본문) 점프와 빠른 명령(새 문서 4종/프로젝트/아카이브/Sonnet AI/설정)을 한 입력으로. ↑↓·⏎·⎋.
- **문서 스냅샷 + 비교 + 복원** — 탭바 시계 버튼. 이름 붙여 찍고, 블록 단위 diff(추가/삭제/변경 색 배지)로 현재와 비교하고, 복원하면 '복원 전 상태'가 자동 보관되며 ⌘Z로도 되돌릴 수 있다. 수동 저장(⌘S) 시 자동 스냅샷 옵션(기본 켜짐, 문서당 자동분 10개 유지).
- **낭독 리허설 (TTS)** 🔊 — 리허설에 스피커 토글. 대사를 캐릭터마다 다른 목소리로 읽어주고(캐스트 순번 고정 배정), 말이 끝나야 다음 블록이 등장한다. 재생 속도 연동, 일시정지/정지 연동. 진행 카운터(n/N), 끝나면 다시 재생(⌘R), 구분선에서 장면 전환 호흡.
- **읽기 전용 뷰어 모드** — 세 에디터 툴바에 잠금 토글(⇧⌘L). 편집·AI·가져오기가 닫히고 열람·내보내기는 유지.
- **리허설 모드** 🎬 — 읽기 전용 시나리오를 ▶︎(⌘R)로 재생하면 블록이 실제 대화처럼 하나씩 등장. '입력 중…' 인디케이터, 자동 스크롤, 0.5×~2×.
- **Quick Look + Finder 통합** — .scen/.scno/.scpa가 Finder에서 단일 문서로 보이고, 스페이스바 미리보기(캐스트 칩+대사/노드 개요/블록 렌더)와 더블클릭 열기를 지원.
- **문서 내 검색 재설계** — 필터 대신 에디터 표준 점프 탐색: 일치 하이라이트, 현재 위치 액센트 링, n/N 카운터, ↑↓ 이동, ⌘F 포커스. 채팅 흐름의 앞뒤 맥락이 유지된다.
- **집필 목표 카드** — 일별 집필 글자 수를 집계해 프로필에 오늘/목표 진행 바, 연속 집필 🔥 스트릭, 목표 달성 인장. 목표는 설정 > 일반(200~5000자).
- **페이지 포커스/타자기 모드** — 설정 > 페이지 토글. 편집 중 블록 외 디밍 + 편집 블록 화면 중앙 고정.
- **PDF 다중 페이지 분할** — 페이지 Export PDF가 블록 높이를 측정해 A4로 나눠 담는다. 'n / N' 푸터, PDFKit 테스트.
- **작가 프로필 사진 원형 크롭** — 설정 '크롭 조정…' 시트. 캐릭터 크롭 UI를 공용 컴포넌트로 추출해 공유.

### 인터페이스
- **설정 재편** — 4탭 → 사이드바 9카테고리(일반/모양/텍스트/시나리오/페이지/마인드맵/파일 아카이브/Sonnet AI/베타). 공통 설정과 기능별 설정을 분리.
- **눈금-렌즈 슬라이더** — 모든 슬라이더를 @uxmateja 리디자인으로 교체: 왼쪽 값, 세로 눈금(채움=액센트), 현재 위치의 유리 렌즈가 주변 눈금을 확대(1.7×/1.9×), 조절 중 값 팝업, 눈금 크로싱마다 520Hz 틱.
- **탭바** — 미저장 탭에 액센트 점(호버 시 닫기 버튼으로 전환), 선택 탭 자동 스크롤, 홈 탭 X 제거, 칩 상단 클립 해제.
- **테마 일관성** — 테마 전환 시 강조색이 테마 고유 액센트로 리셋(이후 선택이 우선), 저장 중/자동 저장 배지가 테마 액센트를 따름, 마인드맵 연결선이 노드를 관통해 비치던 문제 수정(불투명 베이스).
- **단축키 확충** — ⇧⌘L 읽기 전용, ⌥⌘I 인스펙터, ⌘F 문서 내 검색, ⌘R 리허설, ⎋ 정지. 툴팁에 표기.
- 캐릭터 페이지에서 무의미하던 '/' 힌트 숨김, Pretendard 번들 동봉.

- **캐릭터 페이지 v2** — 탭 구조(프로필/노트/관계/갤러리/보이스), 구조화 필드, 프로필 이미지 팝업 패널(원형 크롭 통합), 방사형 관계도, 갤러리+시점 태그, 선택적 보이스 카드(AI 말투 주입)

- **페이지 이미지/표 블록** — 파일 선택·드래그 앤 드롭·URL 임베드, 비율 지정(원본/16:9/4:3/1:1), 클릭 확대, 편집형 표(행/열 추가·삭제), Markdown 왕복 변환
