# v2.0の新機能 — 印章と原稿

インターフェイスが全面的に再構築されました。温かみのある紙のキャンバスに墨色のインクで書き、行動（生成・確定・AI）にだけ朱色の印が押されます。

## 新しい構造
- **統合タイトルバー** — 文書タブ・開いているタブメニュー・保存状態がウィンドウ上部の一列に集まりました。タブはドラッグで並べ替え、⌘1~9で移動します。
- **左のアイコンレール** — ホーム / ファイルアーカイブ / 検索(⌘K) / Sonnet AI / 受信箱。下部に設定とプロフィール。
- **ファイルアーカイブ** — 左のカテゴリー列から種類別に探索します。隠しファイルはTouch IDで保護されます。

## 新機能
- **プロットタイムライン** — シナリオ上部のシーンカードストリップ。カードをドラッグすると本文の順序も変わり、シーンから分かれた分岐チップを押すと分岐に切り替わります。
- **シーンモード** — 入力欄のセリフ/指示/シーンタブ（Tabキーで循環）。シーンモードでタイトルを入力するとシーン境界が挿入されます。
- **文書埋め込み** — ページの / コマンドから他の文書のライブプレビューをブロックとして挿入できます。
- **関係 → マインドマップ昇格** — キャラクターの関係タブから、関係網を新しいマインドマップ文書として作成できます。
- **⌘K 開いているタブ** — パレット最上部から開いているタブへ直接切り替えられます。

## ガイドプロジェクト
- 設定 > ガイドプロジェクトから、チュートリアルプロジェクトを直接ダウンロードできます。アプリの言語に合わせて韓国語/日本語/英語版が用意されています。
