# Changelog

## v1.0 — 2026-07-09

첫 배포 버전. 3종 에디터와 프로젝트 관리 기능의 기본기를 갖췄다.

### 에디터
- **시나리오(.scen)** — 대사/지침 블록, 다중 화자 ⌘선택, 드래그 정렬, 캐릭터 인스펙터, AI 제안 수락/무시
- **시나리오 분기** — 블록 우클릭 '여기서 분기 만들기', 툴바 분기 피커(본편/분기 전환), 분기점 컨텍스트 표시, AI 이어쓰기가 활성 분기 흐름을 따름
- **마인드맵(.scno)** — 팬/줌, 더블클릭 노드 생성, 연결선+캡션, 노드 인스펙터, 이미지 노드
- **블록 페이지(.scpa)** — `/` 커맨드, Enter 분할, 빈 블록 Backspace 병합, Tab 들여쓰기, 토글 접기, Markdown Import/Export
- **캐릭터 페이지 v2** — 탭 구조(프로필/노트/관계/갤러리/보이스), 구조화 필드, 프로필 이미지 팝업 패널(원형 크롭 통합), 방사형 관계도, 갤러리+시점 태그, 선택적 보이스 카드(AI 말투 주입)
- **캐릭터 락업 양방향** — 캐스트→캐릭터 페이지 생성 + 프로젝트 캐릭터 페이지→캐스트 가져오기(프로필 승계)
- **페이지 이미지/표 블록** — 파일 선택·드래그 앤 드롭·URL 임베드, 비율 지정(원본/16:9/4:3/1:1), 클릭 확대, 편집형 표(행/열 추가·삭제), Markdown 왕복 변환
- **페이지 Export** — Markdown + HTML(이미지 base64 임베드) + PDF

### 프로젝트 관리
- **참조 패널** — 문서별 속성/참조(추가·제거·열기)/백링크, 저장 시 캐스트·마인드맵 링크에서 자동 참조 파생
- **본문 검색** — SQLite 색인에 평문 포함, 홈 검색이 제목+본문 딥서치
- **파일 아카이브** — 카테고리/정렬/리스트·그리드, 가리기·휴지통(Touch ID 게이트, Finder 반영)
- **저장** — 자동(디바운스)/수동(⌘S), 4색 상태 배지, 종료 시 자동 백업 + 타임라인 복원
- **설정** — 4탭, 언어(한/일/영) 즉시 전환, 강조색 5종+커스텀, 품질 3단계, 배경 파라미터
- **AI 에이전트 채팅** — 전용 탭 + 사이드패널 미니 챗, 제공자 3종(Apple 온디바이스/Anthropic/오프라인) 공용
- **단축키** — ⌘T/⇧⌘A/⌘W/⌘S/⌘1~9, 슬래시 메뉴 ↑↓·Enter
- **Touch Bar 지원(베타)** — 설정 토글 + 기능 목록/프리뷰 + 실제 NSTouchBar 배선

### 디자인 & 성능
- Sonnet 테마(앤티크 페이퍼 + 적갈 액센트, Pretendard 기본), 깃털 앱 아이콘
- Chrome-tabs 풀폭 탭바(캡슐 옵션), 픽셀 브리딩 필드(홈/사이드바), Liquid Glass 끄기 기본
- 전체화면/윈도우 레이아웃 분기 — 헤더·픽셀 필드가 각각 다른 위치·크기로 렌더링
- 모션 패스 — 카드 호버 리프트+눌림, 저장 배지 펄스, 사이드바 탭 슬라이딩 하이라이트, AI 채팅 메시지 도착 트랜지션
- Wavy Dot Field — Metal 셰이더, 셰이더 부재 시 Canvas 폴백
