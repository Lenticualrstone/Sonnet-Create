# What's New in v2.0 — Seal & Manuscript

The interface has been rebuilt from the ground up. You write in ink on a warm paper canvas, and the vermilion seal appears only on actions — create, confirm, AI.

## New structure
- **Unified titlebar** — document tabs, the open-tabs menu, and save status now live on a single line at the top of the window. Drag tabs to reorder; jump with ⌘1–9.
- **Left icon rail** — Home / File Archive / Search (⌘K) / Sonnet AI / Inbox, with Settings and your profile at the bottom.
- **File Archive** — browse by type in the left category column. Hidden files stay behind Touch ID.

## New features
- **Plot timeline** — a scene-card strip above the scenario. Dragging cards reorders the body itself, and branch chips forked from a scene switch you into that branch.
- **Scene mode** — the composer's Dialogue/Direction/Scene tabs (cycle with Tab). Typing a title in Scene mode inserts a scene break.
- **Embed Document** — the page editor's / command can insert a live preview of another document as a block.
- **Relations → mind map** — promote a character's relationship web into a new mind map document from the Relations tab.
- **⌘K Open Tabs** — switch straight to an open tab from the top of the palette.
