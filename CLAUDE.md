# Remote access policy

This project is kept continuously mirrored to GitHub (`origin/main`,
`Lenticualrstone/Sonnet-Create`) so it can be picked up from claude.ai/code
on any device at any time.

- After any set of code changes in a session, commit them and `git push
  origin main` without waiting for explicit confirmation each time. Use
  clear, conventional commit messages. Do not ask "should I push?" — just do
  it, the same way you would commit-on-request in other projects.
- Exception: if a push would force-push, rewrite history, or the working
  tree contains unrelated/uncommitted work that looks like it belongs to the
  user's own in-progress experiment, stop and ask instead of pushing over it.
- This policy stays in effect until the user removes this section or
  otherwise says to stop auto-pushing.
